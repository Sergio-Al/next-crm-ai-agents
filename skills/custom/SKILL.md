# Custom Skill Template

Create custom skills by following this template. Each skill provides a set of
tools that the agent can invoke during conversations.

## Capabilities

- Describe what this skill enables the agent to do

## Tools

### tool_name
Description of what the tool does.
- **Parameters**: List the input parameters with types
- **Returns**: Describe the return value

## Context

Provide any background context the agent should know when using this skill.
Include guidelines, constraints, and best practices.
